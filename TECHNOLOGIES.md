# 🛠️ Stack Tecnológico - The Warm Embrace

Este proyecto, generado con la precisión de **Lovable**, utiliza una arquitectura moderna de alto rendimiento impulsada por las siguientes tecnologías:

## 🏗️ Core & Lenguaje
- **[React 18](https://react.dev/):** La biblioteca principal para construir la interfaz de usuario basada en componentes modernos.
- **[TypeScript](https://www.typescriptlang.org/):** Para un desarrollo seguro y robusto con tipado estático.
- **[Vite](https://vitejs.dev/):** El motor de construcción y servidor de desarrollo ultrarrápido.

## 🎨 Diseño & Estética
- **[Tailwind CSS](https://tailwindcss.com/):** Framework de CSS basado en utilidades para un diseño ágil y responsivo.
- **[Shadcn UI](https://ui.shadcn.com/):** Colección de componentes reutilizables construidos sobre Radix UI.
- **[Radix UI](https://www.radix-ui.com/):** Primitivas de componentes sin estilo con foco en la accesibilidad (A11y).
- **[Framer Motion](https://www.framer.com/motion/):** Motor de animaciones premium para transiciones fluidas e interacciones dinámicas.
- **[Lucide React](https://lucide.dev/):** Set de iconos modernos y consistentes.
- **[Next Themes](https://github.com/pacocoursey/next-themes):** Gestión inteligente de temas (Dark/Light mode).

## 🚀 Funcionalidad & Datos
- **[TanStack Query](https://tanstack.com/query/latest):** Gestión de estado asíncrono y caché de datos.
- **[React Hook Form](https://react-hook-form.com/):** Manejo eficiente de formularios sin renders innecesarios.
- **[Zod](https://zod.dev/):** Validación de esquemas y tipos en tiempo de ejecución.
- **[React Router DOM](https://reactrouter.com/):** Enrutamiento declarativo para aplicaciones de una sola página (SPA).
- **[Recharts](https://recharts.org/):** Biblioteca de gráficos basada en componentes de React.

## 🧪 Pruebas & Calidad
- **[Vitest](https://vitest.dev/):** Marco de pruebas unitarias compatible con la configuración de Vite.
- **[Playwright](https://playwright.dev/):** Pruebas de extremo a extremo (E2E) robustas y modernas.
- **[ESLint](https://eslint.org/):** Herramienta de análisis estático para garantizar la calidad del código.

## 📦 Herramientas de Desarrollo
- **[Bun](https://bun.sh/):** Ejecutor de JavaScript y gestor de paquetes de alto rendimiento.
- **[PostCSS](https://postcss.org/):** Herramienta para transformar el CSS con plugins.
- **[Autoprefixer](https://github.com/postcss/autoprefixer):** Plugin para añadir prefijos de navegadores automáticamente.
